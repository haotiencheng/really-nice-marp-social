# quote

Large centered pull-quote. Use for memorable lines, book quotes, hot takes.

**Class:** `<!-- _class: quote -->`
**Default background:** `backgrounds/bg-quote.png`

## Example

```html
<!-- _class: quote -->

<div class="quote-wrap">
  <div class="quote-mark">“</div>
  <p class="quote-body">Design the system so that the cheap thing to do is the right thing to do.</p>
  <div class="quote-attr">— Team rule, 2024</div>
</div>
```

**Notes**
- One sentence is ideal. Two-liner max.
- Attribution is optional. Drop `.quote-attr` if none.
