# cta-cards

Vertical stack of compact action cards вҖ” icon badge + title + short meta. Use for multiple open positions, multiple CTAs, "pick one of these".

**Class:** content slide (no `_class`)
**Themes:** `cream-blob`
**Item count:** 2вҖ“4

## Example

```html
# з«ӢеҚіз”іи«Ӣ

<div class="cta-cards">
  <div class="cta-card">
    <div class="icon-badge"><span class="icon">work</span></div>
    <h2>иІЎеӢҷеҜҰзҝ’з”ҹ</h2>
    <p>иҒ·дҪҚ ID: 6299</p>
  </div>
  <div class="cta-card">
    <div class="icon-badge"><span class="icon">campaign</span></div>
    <h2>иЎҢйҠ·жҙ»еӢ•еҜҰзҝ’з”ҹ</h2>
    <p>иҒ·дҪҚ ID: 6457</p>
  </div>
  <div class="cta-card">
    <div class="icon-badge"><span class="icon">analytics</span></div>
    <h2>ж•ёж“ҡеҲҶжһҗеҜҰзҝ’з”ҹ</h2>
    <p>иҒ·дҪҚ ID: 6482</p>
  </div>
</div>
```

**Notes**
- For a QR-based close-out, use `qr-duo` or `closing` instead.
