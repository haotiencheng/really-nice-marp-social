# list-5

Five-bullet list card. Denser variant of `list-3`.

**Class:** `<!-- _class: list-5 -->`
**Default background:** `backgrounds/bg-body.png`

## Example

```html
<!-- _class: list-5 -->

<div class="wrap">
  <div class="eyebrow">5 signals it's time to rewrite</div>
  <div class="card">

    - Every new feature touches the same 3 files
    - Tests rely on ordering, not assertions
    - Every bug report mentions the same module
    - Onboarding time doubles in one quarter
    - You stop saying yes to "small" changes

  </div>
</div>
```

**Notes**
- `list-5` uses tighter padding and smaller line-height than `list-3`.
- If content overflows, drop to 4 bullets or split into two slides.
