# icon-rows

Stacked rows of icon + headline + one-line body. Use for benefits, "why join us", 3–4 feature callouts.

**Class:** content slide (no `_class`)
**Themes:** `cream-blob`
**Item count:** 3–4

## Example

```html
# 為什麼加入 Trip.com

<div class="icon-rows">
  <div class="ir-item">
    <div class="icon-badge"><span class="icon">public</span></div>
    <div class="ir-body">
      <h2>全球舞台</h2>
      <p>國際化團隊、開放包容文化，發揮全球影響力。</p>
    </div>
  </div>
  <div class="ir-item">
    <div class="icon-badge"><span class="icon">stairs</span></div>
    <div class="ir-body">
      <h2>職涯發展</h2>
      <p>領導力、軟技能、專業知識的學習資源。</p>
    </div>
  </div>
  <div class="ir-item">
    <div class="icon-badge"><span class="icon">diversity_3</span></div>
    <div class="ir-body">
      <h2>彈性福利</h2>
      <p>彈性工作安排、內部轉調、全球輪調。</p>
    </div>
  </div>
</div>
```

**Notes**
- Keep body to one line — rows are compact.
- For 5+ items use `timeline` or a second slide.
